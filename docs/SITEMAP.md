## 🎯 페이지별 상세 구조

### 홈페이지 (/)

**섹션 구성:**

1. **Hero Section**

   - 대형 타이포그래피: "신뢰할 수 있는 환경안전 전문기관"
   - 서브카피: "산업보건·먹는물·석면·실내공기질 전문 시험연구기관"
   - CTA 버튼: "서비스 둘러보기", "견적 문의하기"
   - 배경: 고급 실험실 이미지 (parallax 효과)

2. **주요 서비스 카드**

   - 6개 메인 서비스 카드형 레이아웃
   - 각 카드: 아이콘 + 제목 + 간단 설명 + 더보기 버튼
   - Hover 효과: 카드 상승 + 그림자 강화

3. **최신 공지사항**

   - 최근 5개 공지사항 리스트
   - 날짜 + 제목 + 배지(중요/일반)
   - "전체보기" 버튼

4. **빠른 서비스**

   - 성적서 조회
   - 견적 요청
   - 검사 의뢰 신청
   - 문의하기

5. **통계 섹션**

   - 검사 건수 (카운터 애니메이션)
   - 보유 장비 수
   - 연구원 수
   - 누적 고객 수

6. **인증 및 수상**

   - 인증서 로고 슬라이더
   - 공인시험기관 인증 등

7. **오시는 길**
   - 지도 임베드
   - 주소 및 연락처
   - 교통편 안내

## 🔄 사용자 플로우 (User Flow)

### 시나리오 1: 먹는물 검사 의뢰

```
1. 홈페이지 접속
   ↓
2. "먹는물 검사" 메뉴 클릭
   ↓
3. "검사 의뢰 신청 절차" 페이지에서 정보 확인
   ↓
4. 의뢰 신청 폼 작성
   - 시설 정보
   - 검사 항목 선택
   - 시료 채취 희망 일자
   - 연락처
   ↓
5. 신청 완료 (이메일/SMS 알림)
   ↓
6. 관리자 확인 후 일정 조율
   ↓
7. 시료 채취 (방문 or 우편)
   ↓
8. 검사 진행
   ↓
9. 성적서 발급 (이메일 알림)
   ↓
10. 마이페이지에서 성적서 다운로드
```

### 시나리오 2: 성적서 조회 (비회원)

```
1. 홈페이지 접속
   ↓
2. 상단 "성적서 조회" 버튼 클릭
   ↓
3. 성적서 번호 + 비밀번호 입력
   ↓
4. 성적서 조회 결과 화면
   - PDF 다운로드
   - 측정 결과 그래프
```

### 시나리오 3: 관리자 - 공지사항 작성

```
1. 관리자 로그인
   ↓
2. 관리자 대시보드
   ↓
3. "게시판 관리" 메뉴
   ↓
4. "공지사항" 탭 선택
   ↓
5. "새 글 작성" 버튼
   ↓
6. 에디터에서 내용 작성
   - 제목 입력
   - 본문 작성
   - 이미지/파일 첨부
   - 중요 공지 체크
   ↓
7. "발행" 버튼 클릭
   ↓
8. 공지사항 페이지에 게시
```
